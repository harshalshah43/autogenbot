# Steps to run rfq_bot in command prompt
- create following api keys:-
    -geoapify key, 
    -google api key
    -open ai key
- put them in .env file (already existing blank file)
- create venv
- pip install -r req.txt
- run python app.py in rfq_bot folder as current working directory
